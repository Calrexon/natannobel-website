# Rein — Portfolio & Blog

Personal portfolio and blog of **Natan Nobel**

🔗 [reincal.is-a.dev](https://reincal.is-a.dev)

## Tech Stack

- [Nuxt 3](https://nuxt.com) + Vue 3 + TypeScript
- [Tailwind CSS](https://tailwindcss.com)
- [GSAP](https://gsap.com) + [Lenis](https://lenis.darkroom.engineering) smooth scroll
- [Nuxt Content v2](https://content.nuxtjs.org) for blog
- [Sveltia CMS](https://sveltiacms.dev) as headless admin

## Getting Started

```bash
git clone https://github.com/calrexon/reinwebsite.git
cd reinwebsite
pnpm install
pnpm dev
```
